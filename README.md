# Loan Approval Prediction

A machine learning web application that predicts whether a loan application will be approved or not.

Built with Python, Gradio, Pandas, and a trained Machine Learning model.

## Live Demo

🌐 Live App: https://huggingface.co/spaces/darthfarhad/Loan-Approve-Prediction

## Features

- Predict loan approval instantly
- Simple and interactive UI
- Built using Gradio
- Machine Learning based prediction
- Deployable on Hugging Face Spaces

## Input Features

The model uses the following information:

- Gender
- Marital Status
- Dependents
- Education
- Self Employment Status
- Applicant Income
- Coapplicant Income
- Loan Amount
- Loan Amount Term
- Credit History
- Property Area
- Total Income

## Technologies Used

- Python
- Gradio
- Pandas
- Scikit-learn
- Pickle
